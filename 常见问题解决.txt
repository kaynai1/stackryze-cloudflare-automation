【问题 1】运行后自动退出，没有显示任何内容

原因：可能是 Python 未正确安装或依赖包缺失

解决方法：
1. 双击 "修复环境.bat" 自动安装所有依赖
2. 或手动执行以下命令：
   pip install playwright requests
   playwright install chromium

【问题 2】提示 "python 不是内部或外部命令"

原因：Python 未添加到系统 PATH

解决方法：
1. 重新安装 Python
2. 安装时务必勾选 "Add Python to PATH"
3. 或手动添加 Python 到 PATH：
   - 右键 "此电脑" → 属性 → 高级系统设置
   - 点击 "环境变量"
   - 在 "系统变量" 中找到 "Path"
   - 添加 Python 安装路径（如 C:\Python310）

【问题 3】提示 "No module named 'playwright'"

原因：Playwright 未安装

解决方法：
运行以下命令：
pip install playwright
playwright install chromium

【问题 4】提示 "playwright install chromium" 失败

原因：网络问题或权限问题

解决方法：
1. 以管理员身份运行命令提示符
2. 或使用国内镜像：
   pip install playwright -i https://pypi.tuna.tsinghua.edu.cn/simple
   playwright install chromium

【问题 5】运行时提示 "连接超时"

原因：网络连接问题

解决方法：
1. 检查网络连接
2. 确保能访问 https://domain.stackryze.com
3. 检查防火墙设置
4. 尝试使用 VPN

【问题 6】提示 "GitHub 登录失败"

原因：GitHub 账号密码错误

解决方法：
1. 确认 GitHub 邮箱和密码正确
2. 确保 GitHub 账号可以正常登录
3. 如果开启了两步验证，需要使用应用密码

【问题 7】提示 "域名已被注册"

原因：该域名前缀已被其他人注册

解决方法：
换一个域名前缀重试，如：
- myapp → myapp2、myapp123
- blog → myblog、blog123
- api → myapi、api123

【问题 8】提示 "Cloudflare API 错误"

原因：API Key 无效或权限不足

解决方法：
1. 重新生成 API Key
2. 确保权限包含 "Edit zone DNS"
3. 确保 Cloudflare 邮箱正确

【快速修复步骤】

如果以上方法都不行，请按以下步骤操作：

1. 安装 Python：
   https://www.python.org/downloads/
   安装时勾选 "Add Python to PATH"

2. 打开命令提示符（管理员）

3. 运行以下命令：
   pip install --upgrade pip
   pip install playwright requests
   playwright install chromium

4. 重新运行脚本
