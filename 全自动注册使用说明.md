# 全自动 Outlook 注册 v3 使用说明

## 功能特点
- **完全自动化**：随机生成账号密码、姓名、出生日期
- **自动填写**：自动填写所有表单字段
- **验证码解决**：集成第三方验证码服务（可选）
- **无需手动操作**：整个流程自动完成

## 使用步骤

### 1. 准备工作
确保已安装 Python 和依赖：
```bash
pip install playwright requests
playwright install chromium
```

### 2. 运行脚本
双击运行 `全自动Outlook注册v3.bat`

### 3. 选择验证码服务
脚本会提示选择验证码服务：

#### 选项1：使用 2captcha（推荐）
- 访问 https://2captcha.com 注册账号
- 获取 API 密钥
- 选择选项1，输入 API 密钥

#### 选项2：使用 AntiCaptcha
- 访问 https://anti-captcha.com 注册账号
- 获取 API 密钥
- 选择选项2，输入 API 密钥

#### 选项3：不使用验证码服务
- 如果验证码服务不可用，选择选项3
- 脚本会在需要验证码时暂停，手动完成后按 Enter 继续

### 4. 等待注册完成
脚本会自动：
1. 生成随机邮箱和密码
2. 生成随机姓名和出生日期
3. 填写所有表单字段
4. 尝试自动解决验证码（如果配置了服务）
5. 完成注册并保存账号信息

### 5. 查看结果
注册成功后，账号信息会：
- 显示在屏幕上
- 保存到 `outlook_accounts.txt` 文件
- 截图保存为 `outlook_success_*.png`

## 验证码服务配置

### 2captcha 配置
1. 访问 https://2captcha.com
2. 注册账号并充值
3. 获取 API 密钥（在仪表板中）
4. 在脚本中选择选项1，输入密钥

### AntiCaptcha 配置
1. 访问 https://anti-captcha.com
2. 注册账号并充值
3. 获取 API 密钥
4. 在脚本中选择选项2，输入密钥

## 费用说明
- 2captcha：约 $2-3 每千次验证码
- AntiCaptcha：约 $2-4 每千次验证码
- 可以先少量充值测试

## 故障排除

### 问题1：页面加载超时
- 检查网络连接
- 尝试使用 VPN
- 脚本已设置60秒超时

### 问题2：验证码解决失败
- 检查 API 密钥是否正确
- 确保账户有足够余额
- 尝试手动完成验证码

### 问题3：脚本自动关闭
- 运行 `快速诊断.bat` 检查环境
- 确保 Python 和依赖已正确安装

## 文件说明
- `全自动Outlook注册v3.py` - 主脚本
- `全自动Outlook注册v3.bat` - 运行批处理文件
- `outlook_accounts.txt` - 账号信息保存文件
- `outlook_success_*.png` - 注册成功截图
- `outlook_error_*.png` - 错误截图

## 注意事项
1. 请确保网络连接正常
2. 验证码服务需要付费，建议先少量测试
3. 注册成功后请妥善保管账号信息
4. 脚本仅供学习研究使用

## 技术支持
如果遇到问题：
1. 运行 `快速诊断.bat` 检查环境
2. 查看错误截图
3. 检查网络连接和代理设置
