【一、邮箱注册功能】

1. 注册 Outlook 邮箱
   文件：注册Outlook邮箱.py / 注册Outlook邮箱.bat
   功能：自动创建 Outlook/Hotmail 邮箱账号
   使用：双击运行，按提示操作

【二、域名注册功能】

2. 使用已有账号注册域名
   文件：一键注册.py / 点击运行.bat
   功能：使用已有 GitHub 账号注册 Stackryze 域名并托管到 Cloudflare
   使用：双击运行，输入账号密码

3. 全自动注册（推荐）
   文件：全自动注册.py / 全自动注册.bat
   功能：自动完成所有步骤
     * 自动注册 Outlook 邮箱
     * 用 Outlook 邮箱注册 Stackryze
     * 注册免费域名
     * 托管到 Cloudflare
   使用：双击运行，输入 Cloudflare 信息

【三、环境工具】

4. 快速诊断
   文件：快速诊断.py / 快速诊断.bat
   功能：快速检查环境问题
   使用：双击运行

5. 修复环境
   文件：修复环境.bat
   功能：自动安装所有依赖
   使用：双击运行

6. 测试环境
   文件：测试环境.py
   功能：详细测试环境配置
   使用：双击运行

   功能：Outlook 邮箱注册详细说明

10. 问题解决指南.txt
    功能：问题解决指南

11. 功能列表.txt
    功能：本文件，列出所有功能

【快速开始】

新手推荐：
1. 双击 "快速诊断.bat" 检查环境
2. 双击 "修复环境.bat" 安装依赖
3. 双击 "全自动注册.bat" 开始注册

已有 GitHub 账号：
1. 双击 "点击运行.bat"
2. 输入 GitHub 账号密码
3. 输入 Cloudflare 信息

只需要 Outlook 邮箱：
1. 双击 "注册Outlook邮箱.bat"
2. 按提示操作

【推荐使用流程】

第一次使用：
1. 双击 "快速诊断.bat"
2. 如果有问题，双击 "修复环境.bat"
3. 双击 "全自动注册.bat"
4. 按提示完成注册

日常使用：
1. 双击 "全自动注册.bat"
2. 输入信息
3. 等待完成

【注意事项】

1. 网络要求
   - 需要能访问国际网站
   - 建议使用稳定的网络

2. 系统要求
   - Windows 10/11
   - Python 3.8 或更高版本
   - 足够的磁盘空间

3. 账号安全
   - 生成的账号信息会保存到文件
   - 请妥善保管账号密码
   - 建议定期备份账号信息

4. 使用限制
   - Stackryze 每个账号最多 6 个域名
   - Outlook 注册可能有频率限制
   - 请合理使用
